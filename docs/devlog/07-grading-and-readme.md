## 핵심 발견 (talking point)

**두 함정의 성격이 다르고, 백엔드가 갈리는 지점도 다르다.**

- **Q4(코퍼스밖 환각 유도)**: 세 백엔드 전원 "자료에 없음"으로 회피(9·10·10). 저가
  로컬도 통과 → RAG 프롬프트의 "없으면 없다고 하라"가 작동. 코퍼스 grep으로
  ESS 회수기간 수치 0건 직접 확인.
- **Q3(오탐 유도, 정상 생산패턴을 이상으로 착각)**: **openai만 함정에 빠짐(3점)** —
  정상 생산라인을 "이상 신호"로 단정하고 부하이동 권고. claude(10)·로컬 exaone(9)은
  "정상 운영 패턴"으로 정확 판단.

> 결론: **환각 회피(없는 걸 안 지어냄)는 셋 다 잘하나, 오탐 회피(있는 걸 과잉해석
> 안 함)는 갈린다.** "큰 격차=이상"이라는 그럴듯한 직관에 저항하려면 도메인 추론이
> 필요하고, 여기서 opus와 의외로 exaone 로컬이 gpt-4o를 앞섰다. **오경보 비용이 큰
> 진단 현장에선 이 한 번의 오진이 opus 프리미엄(약 5배 비용, +1.7점)의 근거** —
> 단순/저위험 질의는 gpt-4o나 로컬로 라우팅.
